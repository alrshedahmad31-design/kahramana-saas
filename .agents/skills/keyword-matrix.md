## Dish Page Keywords — Pattern Rules

### Title Pattern
AR: `[اسم الطبق] — كهرمانة بغداد | مطعم عراقي البحرين`
EN: `[Dish Name] — Kahramana Baghdad | Iraqi Restaurant Bahrain`

### Meta Description Pattern
AR: `[وصف قصير للطبق] في كهرمانة بغداد، مطعم عراقي أصيل في [الرفاع / قلالي]، البحرين. اطلب الآن عبر واتساب.`
EN: `[Short dish description] at Kahramana Baghdad, authentic Iraqi restaurant in [Riffa / Qallali], Bahrain. Order via WhatsApp.`

### Competitive Keyword Logic
Dish page for "مسكوف" competes for:
- مسكوف البحرين
- سمك مشوي البحرين (non-Iraqi search → captures expat/local market)
- grilled fish Bahrain (English search)
- masgouf restaurant Bahrain

Dish page for "كباب" competes for:
- كباب البحرين (ANY kebab search)
- kebab Bahrain (English search)
- BBQ Bahrain

This is the core strategy: each dish page ranks for both the Iraqi AND the generic version.

## Competitor Keyword Capture

Kahramana's 175+ dish pages can capture searches people make for ANY restaurant:
- Searches for specific dishes (masgouf, qoozi, kebab) → Kahramana dish page appears
- Generic food searches (grilled meat Bahrain, fish restaurant Bahrain) → category pages
- Location-based (restaurant Riffa, restaurant Qallali) → branches page

This means the SEO strategy is not "rank for Iraqi food" — it is "rank for FOOD in Bahrain."
