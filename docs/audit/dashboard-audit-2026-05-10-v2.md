## Top 5 Critical Fixes

1. Fix analytics branch-scope leaks across `/dashboard/analytics/**`: make all helper queries branch-aware or owner/GM-only, especially branch comparison, customer, marketing, menu, and operations panels.
2. Fix `/dashboard/schedule` read queries so branch managers cannot see staff, shifts, or leave counts outside their branch.
3. Make waiter branch scope fail closed in `createWaiterOrder`; scoped roles with missing `branch_id` must be rejected.
4. Move purchase order creation into a validated transaction/RPC so bad item payloads cannot leave empty or partial POs.
5. Add server-side transition matrices and row-count checks for reservations, waitlist, catering, and purchase order status changes.
